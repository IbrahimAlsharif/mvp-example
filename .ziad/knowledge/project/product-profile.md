# Product Profile — Human Timeline Network

**Tagline:** "Netflix for human lives and family history."
**What it is:** A living timeline of a person's life, memories, relationships, and
legacy across mobile (iOS/Android) and web. Privacy circles, multimedia memories,
future capsules, and cross-generational inheritance.
**Stage:** Early product / MVP.
**Owner:** WE WILL TECH.

## MVP critical assumption (under test)
Will people **continuously create and revisit** life events inside a dedicated
timeline product?
- **Primary metric:** Events Added Per User Per Month.

## Four core values (from Value-Quality Map)
1. **Preserve** — capture life durably, never lose it.
2. **Navigate** — revisit and re-experience a life over time (browse + playback).
3. **Share with trust** — control who sees each memory (privacy circles).
4. **Legacy across generations** — outlive the owner (future capsules + inheritance).

## High-impact features (🔴)
- Timeline Event Creation (the exact action the metric counts)
- Multi-media capture & upload (photo / video / voice)
- Privacy Circles (Me Only / Family / Public)
- Life Timeline browsing (year / month / day)

## Top domain risks (analyst lens)
- **Wrong-circle exposure** — private item shown publicly → irreparable trust loss, GDPR/PII liability.
- **Permanent data loss** — irreplaceable memories lost → catastrophic, unrecoverable churn.
- **Silent upload drops** — event looks saved, media never persists → kills the primary metric.
- **Future Capsule fails to unlock / no inheritance handoff** → breaks the central legacy promise.
- **Child-safety & CSAM exposure** in the Public circle.

## Analyst takeaway
The product's wedge is the **unified system** (all four values together), and the
**continuous-habit** bet — areas where incumbents are weakest. Differentiation is
NOT future capsules (table stakes) but **trustworthy privacy + a recurring
event-creation habit no competitor has proven.**
