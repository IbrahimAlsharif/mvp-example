# Trustworthy — Competitor Profile (VERIFIED)

Not a memory-timeline app — a **documents/estate vault** — but the single most
important benchmark in the set because it is the **only competitor that enforces
privacy at the asset level and ships a real after-death legacy handoff.** It shows
what "trust" looks like when it is architecture, not UI.

## What it does (analyst workflow it enables)
Security-first encrypted vault for a family's critical info (IDs, money, property,
passwords, legal, emergency), with advisor/executor collaboration and a defined
after-death legacy-access process.

## Core design pattern (not the feature surface)
**Privacy is enforced at the data layer, not promised in the UI.** AES-256 in transit
+ at rest, keys derived from account + device passcode, near-zero-knowledge via
**irreversible aliasing** ("cannot be solved even by the Trustworthy team"). This is
the architectural answer to HTN review heuristic **H-A1** ("the leak is the URL, not
the circle UI") — access is gated at the object, behind MFA, by design.

## Verified facts
- **Platforms:** Web, iOS, Android (all plans, offline mode). `official`
- **Security:** AES-256 (transit+rest); 2FA on by default; biometric; **YubiKey
  hardware-key support** (claims sole family-info platform to offer it); **SOC 2 Type 2,
  SOC 3, HIPAA**, "meets all GDPR controls," CCPA service provider; never sells/shares;
  doesn't train AI on customer data; export anytime; irreversible deletion on cancel. `official`
- **Legacy (strongest in the set):** "Legacy Access Collaborators" — after death, a
  collaborator completes identity verification (gov ID + **death certificate**),
  Trustworthy validates, then grants access. A real, auditable handoff — not a
  scheduled message. `official`
- **Pricing:** Free ($0, 2GB); Silver $10/mo; Gold $20/mo (unlimited, AI); Platinum
  $40/mo (concierge) — billed annually; 50% off military/first responders. `official`
- **Funding:** **$15M Series A (Apr 2022), Valor Siren Ventures; ~$19.7M total.** `analysis`

## What HTN should benchmark against
- **Asset-level access enforcement + MFA** as the standard HTN's privacy circles must
  meet to be more than theater (directly answers H-A1).
- **Death-certificate-verified legacy handoff** as the credible alternative to vague
  "trusted contact" triggers that the rest of the legacy cluster hand-waves.
- Compliance posture (SOC 2 T2, HIPAA) as the enterprise/trust bar.

## Known gaps (HTN's openings)
- **No multimedia memory archive, no life timeline, no resurfacing/playback.** It stores
  documents, not a life you re-experience.
- **No continuous-capture habit** — a vault you set up once, not a weekly memory ritual.
- Document/utility framing has none of the emotional pull HTN's memory content carries.

## Analyst takeaway
Trustworthy is proof that **enforceable privacy + verified legacy handoff is buildable
and fundable** — and that no one has applied it to *multimedia family memories*. HTN's
clearest defensible wedge is to bring Trustworthy-grade enforcement (object-level access,
verified inheritance) to a navigable, emotional memory timeline — the exact intersection
Trustworthy (cold document vault) and Tinybeans/FamilyAlbum (UI-only memory albums) each
leave open.
