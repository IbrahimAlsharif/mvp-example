# References — analytic-methods/

Generic analytic tradecraft and methodologies. These are domain-agnostic
frameworks that apply across intelligence and analysis disciplines.

## What belongs here

- Collection and source evaluation frameworks (CRAAP, SIFT, etc.)
- Structured analytic techniques (ACH, Devil's Advocate, Red Team)
- Intelligence product types and briefing formats
- Intelligence cycle documentation
- Cognitive bias patterns relevant to analysis
- Multi-source fusion and contradiction resolution methods

## These files are slow to go stale

Analytic methods change on the scale of years, not days. A `review_after` of
30–90 days is appropriate. Update only when a major methodology shift occurs
or a new technique becomes standard practice.

## Relationship to domain/ files

`analytic-methods/` contains the generic version. If you need a
client-calibrated version of a method, create it in `references/domain/` and
note which general file it supersedes.
